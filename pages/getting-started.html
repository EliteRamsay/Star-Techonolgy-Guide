Getting Started

Welcome to Star Technology! This section will guide you on install the modpack and getting a world started.

  1. Install the Modpack

    You can install the Modpack on CurseForge

  2. Ram Allocation

    It is recommended to allocate at least 8GB of RAM to the modpack for optimal performance. You can do this in the CurseForge launcher settings.

  3. Creating a New World

    When creating a new world, it is recommended to use the following settings:

    • World Type: Default
    • Generate Structures: off
    • World Seed: (Optional) You can use a custom seed or leave it blank as you will always spawn on a floating island.
    • Game Mode: Survival
    • Difficulty: Peaceful
    • Bonus Chest: off
    • Cheats:off

  4. First Steps

    Upon starting your world, you will spawn on top a tree on a floating island. It is recommended to break the leaves first then the wood so no saplings are lost. To gather stone shift + right click on any grass block to get stone pebbles. which can be crafted into stone with for of them in a square shape.
